Request (Requisição)

O protocolo HTTP (Hypertext Transfer Protocol)
é um conjunto de regras que permite a comunicação entre servidores e clientes

  • Método HTTP (HTTP Method) Define o tipo de ação a ser executada:
  1. GET – Buscar dados.
  2. POST – Enviar/registrar dados.
  3. PUT – Atualizar dados existentes.
  4. DELETE – Remover dados.

O cliente manda um Request (Requisição) e quando o servidor
manda um response (resposta) que são codigos como 200 de
sucesso e 500 como erro.
Isso pode ser observado no Network na ferramente Dev (f12)
