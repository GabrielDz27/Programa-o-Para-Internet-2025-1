Request (Requisição)

O protocolo HTTP (Hypertext Transfer Protocol)
é um conjunto de regras que permite a comunicação entre servidores e clientes

  • 1. Método HTTP (HTTP Method) Define o tipo de ação a ser executada:
  1. GET – Buscar dados.
  2. POST – Enviar/registrar dados.
  3. PUT – Atualizar dados existentes.
  4. DELETE – Remover dados.
